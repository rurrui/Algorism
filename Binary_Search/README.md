#Note 
尽管二分查找的项列表远远大于顺序查找法，但是时间却短短小于普通顺序查找法  
说明二分查找法远远优于顺序查找法  
最差的复杂度：  
普通顺序查找：O(N)  
二分查找法：O(log2(N))
